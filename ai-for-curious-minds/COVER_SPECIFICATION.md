## KDP COVER CREATOR INSTRUCTIONS

### Step 1: Access Cover Creator
1. Go to your KDP Bookshelf
2. Click on the book or "Create New"
3. In Cover section, select "Launch Cover Creator"

### Step 2: Choose Template
- Select a template with similar layout (title prominent, author bottom)
- Or start with "Blank" for full control

### Step 3: Set Background
- Use solid color: Navy blue (#1B2838)
- Or upload a subtle gradient image

### Step 4: Add Front Cover Text
1. **Title:** "START HERE:" (line 1), "AI for CURIOUS MINDS" (line 2)
2. **Subtitle:** "A Beginner's Guide to Artificial Intelligence"
3. **Author:** "Idowu J Gabriel, Sr."

### Step 5: Add Graphic
- Upload the AI brain icon (if you have it as PNG)
- Or use Cover Creator's built-in tech/AI graphics
- Position center-lower on front cover

### Step 6: Add Back Cover Text
- Copy text from this spec (corrected versions)
- Leave barcode area clear (KDP adds automatically)

### Step 7: Preview & Download
- Use Preview to check all text is within safe zones
- Download PDF for review before finalizing
