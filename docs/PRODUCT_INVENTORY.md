## Product Variants: AIXORD Family

### AIXORD: AI Execution Order Framework (Main)
- **Target:** Claude Code Pro users
- **Format:** Full methodology + 8 templates
- **Requirements:** Claude Pro subscription, folder access
- **Price:** $9.99

### AIXORD Chatbot Edition
- **Target:** ChatGPT/Gemini free users
- **Format:** Single-file template system
- **Requirements:** File upload capability only
- **Key Differences:**
  - Token-aware handoff tracking
  - All context in one uploadable file
  - No folder structure needed
  - Simpler keyword commands
- **Price:** $4.99-$7.99

### AIXORD Visual Audit Playbook
- **Target:** QA teams, platform owners
- **Format:** VA-AIXORD methodology
- **Use Case:** End-to-end visual platform audits
- **Includes:** Gap classification, severity scales, audit templates
- **Price:** $9.99

## Professional Service Product Template

Each professional assistant follows the same architecture:

### Keyword-Triggered Workflows
```
User uploads documents + enters keyword
         ↓
Keyword triggers specific workflow
         ↓
AI guides through gated steps
         ↓
Quality checkpoints with evidence
         ↓
Deliverable generated
```

### Example: Tax Assistant Keywords

| Keyword | Workflow |
|---------|----------|
| 1040 | Personal income tax return |
| 1099 | Freelancer/contractor taxes |
| SCHEDULE-C | Small business income |
| AMENDED | Correcting a filed return |
| AUDIT-RESPONSE | IRS audit assistance |

### Example: Legal Assistant Keywords

| Keyword | Workflow |
|---------|----------|
| LLC | Business formation |
| NDA | Non-disclosure review/drafting |
| LEASE-REVIEW | Rental agreement analysis |
| TRADEMARK | Trademark application |
| DEMAND-LETTER | Collections/dispute letter |
