# PMERIT Product Brand Style Guide

**Version:** 1.0
**Created:** December 25, 2025
**Purpose:** Maintain consistent visual branding across all PMERIT products

---

## Cover Design Standards

### Color Palette

| Element | Color | Usage |
|---------|-------|-------|
| **Primary Background** | Dark Blue (#0A1628 or similar) | Main cover background |
| **Accent** | Teal/Cyan (#00D4FF or similar) | Highlights, icons, borders |
| **Text Primary** | White (#FFFFFF) | Titles, author name |
| **Text Secondary** | Light Gray/Silver | Subtitles, taglines |

### Typography

| Element | Style |
|---------|-------|
| **Product Name** | Bold, ALL CAPS, large |
| **Subtitle** | Regular weight, smaller |
| **Tagline** | Italic or light weight, positioned below subtitle |
| **Author** | Regular, bottom of cover |
| **PMERIT Logo** | Top left corner with "+" icon |

### Layout Structure

```
┌─────────────────────────────────────┐
│  + PMERIT                    [top]  │
│                                     │
│     [PRODUCT NAME]                  │
│     [Large, Bold, Caps]             │
│                                     │
│     Subtitle Line                   │
│     Methodology/Description         │
│                                     │
│     "Tagline in quotes"             │
│                                     │
│                                     │
│     Author Name              [bot]  │
└─────────────────────────────────────┘
```

### Visual Elements

- **Icon/Graphic:** Centered, relevant to topic (brain for AI, code brackets for dev)
- **Border/Accent Line:** Teal accent line or border element
- **Clean spacing:** Generous margins, uncluttered design

## Brand Voice

**Tone:** Professional, accessible, empowering

**Key Messages:**
- Democratizing knowledge
- AI-augmented, human-directed
- Practical, actionable methodology
- Built by a solo developer, for solo developers

**Positioning:**
- NOT "AI replaces you"
- YES "AI augments you"
- Focus on methodology over magic
