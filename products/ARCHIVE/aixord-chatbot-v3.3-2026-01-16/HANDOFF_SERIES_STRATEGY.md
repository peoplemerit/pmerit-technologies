## 5. Execution Phases

### Phase 1: Create Gumroad Discount Codes (Manual — Director)

1. Log into Gumroad (meritwise0.gumroad.com)
2. For EACH of the 8 products:
   - Go to Product → Discount Codes
   - Create new code with product-specific name
   - Set to 100% off
   - Limit to that product only
3. Delete `AIXORD2025BOOK` code

```bash

```

### Phase 4: KDP Upload (Manual — Director)

For each book:
1. Go to kdp.amazon.com
2. Bookshelf → Create → Kindle eBook
3. Enter metadata from Section 3 of this HANDOFF
4. Upload corresponding DOCX
5. Upload/create cover
6. Set pricing per Section 3
7. Add to "AIXORD Framework Series"
8. Publish

### Phase 5: Verify End-to-End (Manual — Director)

For each book:
1. Purchase book (or use preview)
2. Verify discount code is visible in Appendix B
3. Test code on Gumroad → should work ONLY for that product
4. Confirm download works
