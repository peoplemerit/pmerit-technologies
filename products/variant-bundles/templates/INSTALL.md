# AIXORD for {{VARIANT}} — Installation Guide

## Quick Install (4 steps)

1. **Unzip** this file to a folder on your computer
2. **Open Chrome** → Go to `chrome://extensions`
3. **Enable Developer Mode** (toggle in top-right)
4. **Click "Load unpacked"** → Select the `extension/` folder

## What's Included

- `extension/` — AIXORD Companion Chrome extension
- `manuscript/` — AIXORD governance documentation (if included)
- `variant.json` — {{VARIANT}}-specific configuration

## Using AIXORD

1. Go to {{PLATFORM_URL}}
2. Click the AIXORD icon in your toolbar
3. Side panel opens with gate tracker and phase selector
4. Follow the AIXORD workflow:

   **Setup Phase (6 gates)**
   - LIC → License acknowledgment
   - DIS → Disclaimer acceptance
   - TIR → Tier selection
   - ENV → Environment setup
   - OBJ → Objective definition
   - RA → Reality assessment

   **Work Phase (7 gates)**
   - FX → Formula application
   - PD → Project docs review
   - PR → Plan review
   - BP → Blueprint creation
   - MS → Master scope
   - VA → Visual audit
   - HO → Handoff

## Keyboard Shortcuts

- `Alt+A` — Open/close side panel
- `Alt+G` — Quick gate toggle (when panel open)

## Troubleshooting

**Extension not loading?**
- Make sure you selected the `extension/` folder, not the parent folder
- Check that Developer Mode is enabled

**Side panel not appearing?**
- Click the puzzle piece icon in Chrome toolbar
- Pin the AIXORD Companion extension

**Gates not saving?**
- Extension data is stored locally in Chrome
- Clearing browser data will reset progress
