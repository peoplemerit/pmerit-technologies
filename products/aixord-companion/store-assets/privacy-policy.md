# AIXORD Companion Privacy Policy

**Last Updated:** January 25, 2026

## Overview

AIXORD Companion is a Chrome extension developed by PMERIT Technologies LLC. This privacy policy explains how we handle your data.

## Data Collection

**AIXORD Companion does NOT collect, transmit, or store any personal data.**

The extension:
- Does NOT track your browsing activity
- Does NOT collect analytics or telemetry
- Does NOT require user accounts
- Does NOT send data to external servers

## Local Storage

The extension stores the following data locally in your browser:
- Gate completion status
- Current phase selection
- Session notes you enter

This data:
- Never leaves your device
- Is stored only in Chrome's local storage
- Can be cleared by removing the extension

## Permissions

The extension requests the following Chrome permissions:

- **sidePanel**: To display the governance panel alongside your tabs
- **storage**: To save your gate and phase progress locally
- **activeTab**: To detect which AI platform you're using (for variant configuration only)

## Third-Party Services

AIXORD Companion does not integrate with or send data to any third-party services.

## Optional Web App Integration

If you choose to use the AIXORD Web App (https://aixord-webapp-ui.pages.dev), that service has its own privacy policy. The Companion extension can operate fully offline without the web app.

## Data Deletion

To delete all data stored by AIXORD Companion:
1. Right-click the extension icon
2. Select "Remove from Chrome"
3. All local data is automatically deleted

## Changes to This Policy

We may update this privacy policy. Changes will be noted with a new "Last Updated" date.
