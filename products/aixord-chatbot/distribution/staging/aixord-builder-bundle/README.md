## Quick Start

### Starting a New Project
1. Open any AI
2. Paste `AIXORD_GENESIS.md`
3. Describe your idea
4. Let the project evolve

### Continuing a Project
1. Open any AI
2. Paste `AIXORD_UNIVERSAL_ENHANCED.md`
3. Paste your last HANDOFF
4. Say `CONTINUE`

### Decomposing into SCOPEs
When your project has 3+ components:
1. Create `MASTER_SCOPE.md` using template
2. Create individual `SCOPE_*.md` files
3. Track progress per-scope

| Command | Effect |

1. MASTER_SCOPE as project contract
2. SCOPEs as deliverables
