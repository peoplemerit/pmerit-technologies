## Choose Your Variant

### Gemini Advanced

Leverages Gemini's unique capabilities:
- **1M token context** — Load entire project documentation
- **Gems** — Create persistent AIXORD assistant
- **Google Workspace integration** — Store handoffs in Drive

### Gemini Free

Optimized for free tier:
- Condensed governance
- Manual paste per session
- Google Drive for handoff storage

## Quick Start (Gem Method)

1. Open Gemini → Gem Manager → Create Gem
2. Name: "AIXORD"
3. Paste variant instructions
4. Save

Your Gem now has AIXORD governance built in.

---

## Quick Start (Paste Method)

1. Open gemini.google.com
2. Start new conversation
3. Paste contents of your variant file
4. AI acknowledges AIXORD
5. Work with structured governance

## Using Google Drive

Store handoffs in Drive for easy access:

1. Create folder: `AIXORD-Projects/[ProjectName]/`
2. Save handoffs: `HANDOFF_S1.md`, `HANDOFF_S2.md`
3. Reference: "Check my Drive for the last handoff"
