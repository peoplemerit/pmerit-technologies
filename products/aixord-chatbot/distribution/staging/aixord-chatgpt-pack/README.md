## Quick Start (Custom GPT Method)

1. Go to ChatGPT → Explore GPTs → Create
2. Name: "AIXORD Architect" (or your preference)
3. Paste variant instructions into GPT Instructions
4. Save and use

Your Custom GPT now has AIXORD governance built in.

---

## Quick Start (Paste Method)

1. Start new ChatGPT conversation
2. Copy entire contents of your variant file
3. Paste as first message
4. AI acknowledges AIXORD
5. Work with structured governance

## Using Memory (Plus/Pro)

ChatGPT Plus and Pro have Memory. Use it:

```
"Remember: I use AIXORD governance. Current project is [name].
Last decision was D-003 (Use React). Active scope is SCOPE_AUTH."
```

ChatGPT remembers across sessions.

## Using Extended Thinking (Pro)

For complex architectural decisions:

```
"Think through this carefully: Should we use microservices
or monolith? Consider team size, scale, and budget."
```

ChatGPT Pro will use extended reasoning.
