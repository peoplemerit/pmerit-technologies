## Quick Start

### Step 1: Choose Your Variant
- Starting from idea? → Genesis
- Claude Pro + Code? → Claude Dual
- ChatGPT Plus? → ChatGPT Plus
- Any AI? → Universal Enhanced

### Step 2: Paste Governance
Copy your chosen variant into your AI session.

### Step 3: Work
Use DECISION mode for planning, EXECUTION mode for building.

### Step 4: Save Progress
Say `HANDOFF` at session end. Save the output.

### Step 5: Continue
Paste HANDOFF at next session start. Say `CONTINUE`.
