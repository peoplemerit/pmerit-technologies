# Product Stock

This directory contains all finalized PMERIT products ready for distribution.

## Directory Structure

```
Product-Stock/
├── Books/                    # Published books (KDP, Gumroad, etc.)
│   ├── [book-name]/
│   │   ├── FINAL_INTERIOR.pdf
│   │   ├── COVER.pdf
│   │   └── METADATA.md
│   └── ...
├── Courses/                  # Video courses, learning paths
├── Templates/                # Notion, spreadsheet, document templates
└── Digital-Downloads/        # Standalone digital products
```

## Naming Convention

- Use lowercase with hyphens: `ai-for-curious-minds`
- Final files should be prefixed with `FINAL_`
- Keep source files in the development folder, only finals here

## Product Status Key

| Status | Meaning |
|--------|---------|
| DRAFT | In development |
| REVIEW | Ready for final review |
| FINAL | Approved, ready for distribution |
| LIVE | Published and available for sale |
